# RetroVibe
RetroVibe is a web-based application designed to fetch and display real-time data from any YouTube channel using the official YouTube Data API v3.

**steps to use it.**
1.Launch: Open the web application (the Vercel link).\
2.Input: In the “Channel URL/Handle” box, type the handle of the creator you want to look up (e.g., @MrBeast or @MKBHD).\
3.Submit: Click the “Submit” button.\
4.View: The app will instantly display the Channel Name, Subscriber Count, and Total Views in a glowing results box.

**Use of AI**
AI is used for connection between the frontend and backend that is between the connection of flask and index.html
